# 课程学习平台

一个现代化的在线课程学习平台，支持视频播放、进度跟踪、章节导航和笔记功能。

## 功能特性

✅ **视频播放**
- 支持多种视频格式（MP4, WebM）
- 响应式设计，完美适配PC和移动端
- 自定义播放控制界面
- 播放速度调节（0.5x - 2x）
- 防复制保护（禁用右键菜单和下载控件）

✅ **学习进度跟踪**
- 自动保存学习进度
- 支持断点续学
- 章节完成状态跟踪
- 可视化进度条显示

✅ **章节导航**
- 清晰的章节列表
- 当前章节高亮显示
- 学习状态图标（未开始/进行中/已完成）
- 快速章节切换

✅ **笔记功能**
- 每章节独立笔记
- 自动保存到浏览器本地存储
- 支持富文本编辑
- 未保存更改提醒

✅ **响应式设计**
- 完美适配桌面端和移动端
- 触摸友好的控制界面
- 自适应布局

## 技术栈

- React 18
- React Router 6
- Vite 构建工具
- 纯CSS实现响应式设计
- LocalStorage 本地存储

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 课程列表页面
- 显示所有可用课程
- 显示每个课程的学习进度
- 点击"开始学习"或"继续学习"进入课程

### 视频播放页面
- 左侧：视频播放器
- 右侧：章节导航和笔记面板
- 支持键盘快捷键和触摸操作

### 进度跟踪
- 观看超过90%的视频自动标记为完成
- 进度实时保存到浏览器本地存储
- 支持跨设备同步（需要后端支持）

### 笔记功能
- 点击笔记按钮打开笔记面板
- 笔记自动保存到本地存储
- 支持清空和手动保存

## 浏览器支持

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 部署

1. 构建项目：`npm run build`
2. 将 `dist` 目录部署到Web服务器
3. 配置服务器支持HTML5 History模式
